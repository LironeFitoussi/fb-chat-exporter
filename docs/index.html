Chrome Extension

Export Messenger chats to JSON, locally in your browser.

Messenger Chat Exporter helps you save the currently open Facebook Messenger conversation as a JSON file and keep a local export history for quick re-downloads.

Privacy Policy View Source

What it does

  • Exports the active Messenger conversation as formatted JSON.
  • Stores successful exports in browser storage for later re-downloads.
  • Runs locally in the browser with no external backend.

How it works

Open a conversation in Messenger, launch the extension, and export. The extension reads the messages visible in the current chat and packages them into a downloadable JSON file.

Permissions

The extension requests tabs, downloads, and storage so it can access the active Messenger tab, save the export file, and remember prior exports in your browser.

Privacy first

Exported chat content is processed locally in your browser session. The project does not run a server and does not transmit your conversations to a remote database.
