Messenger Chat Exporter

Effective date: April 24, 2026

Overview

Messenger Chat Exporter is a Chrome extension that lets a user export the currently open Facebook Messenger conversation to a JSON file and keep a local export history in the browser for later re-downloads.

Data the extension accesses

The extension may access the following information only when you use it:

  • Message content visible in the currently open Messenger conversation
  • Conversation metadata needed to build the JSON export
  • Local export history saved in browser storage

How data is used

The extension uses accessed data only to:

  • Read the active Messenger conversation from the current tab
  • Generate a downloadable JSON export file
  • Store export records locally in your browser so you can re-download them later

Data sharing

The extension does not sell your data. It does not send your Messenger conversations to the developer's servers, to external analytics providers, or to third-party advertising services.

Data storage

Export records are stored locally using the browser's extension storage. Downloaded JSON files are saved by the browser to the location you choose or to your configured downloads folder.

Permissions explanation

  • tabs: used to identify and interact with the active Messenger tab
  • downloads: used to save generated JSON files to your device
  • storage: used to remember prior successful exports locally

Data retention and deletion

You control downloaded export files on your device. You can remove locally stored export history by clearing the extension's storage or uninstalling the extension.

Children's privacy

This extension is not directed to children under 13, and the developer does not knowingly collect personal information from children.

Changes to this policy

This privacy policy may be updated if the extension's functionality or data practices change. Updates will be posted on this page.
